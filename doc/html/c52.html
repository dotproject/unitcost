Chapter 1. Technical Documentation

1.1. Versions

Although most of the module is a standard snap-in to the dotProject platform, it is inevitable to patch a few of the original distribution's files. Because of this, the Unitcost module was not built to be downward-compatible with versions of dotProject prior to version 2.0.3.

1.1.1. Current dotProject Supported Versions

2.0.3

1.1.2. Information on Newer Versions

Work for dotProject version 2.0.4 will start in the next few weeks and not expected to be released until the end of August.
