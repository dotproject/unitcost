1.5. Database Changes

Unicost adds two new tables and alters the definition of some original fields. You may un-install the module and these new tables will be removed. Nevertheless the table alterations to the original tables will not be undone for obvious reasons, but on the other hand, all the alterations are basically harmless to the standard dotProject. Check the setup.php file in the unicost module directory for details on the new tables and the alters performed on the original tables.
