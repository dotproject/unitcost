1.3. Post-Installation and Configuration

The Unicost module has been tested to work with your current configuration and even with your current data. Nevertheless, since it adds a whole new perspective to the Task Log capability, we recommend that you:

  1. Edit User Roles or Security to disable the delete option in Task Logs application, even for the Administrator.

  2. Edit User Roles or Security to disable the edit option in Task Logs application, even for the Administrator.

  3. Change the TaskLogReference value list for actions that are appropriate for your industry. For example, in construction it makes sense to change these value for something like: 

    		0|Not Defined 1|Advance Report 2|Valuation 3|Invoice 4|Payment

The reasons for these recommendation are better explained later in this document, but for now take our word for it and follow them, or read this complete document before you decide not to.
